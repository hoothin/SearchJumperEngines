#### Usage

1.  In the SearchJumper extension, go to the **Export Config** page.
2.  Click **Import Config**.
3.  Select the desired `.json` file from this repository to import.

#### Contributing

If you have a website or a useful tool you'd like to add, contributions are welcome. The goal is to add new search engines to the existing JSON files.

1.  Fork this repository.
2.  Choose the appropriate category (`AI`, `Image`, or `Misc`) and language (`_en`, `_tw`, or `_zh`).
3.  **Modify the corresponding `.json` file** to add your search engine entry. Please maintain the existing format.
4.  Submit a pull request with a brief description of your changes.
